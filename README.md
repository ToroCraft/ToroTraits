## Development Environment Setup

```
git clone git@github.com:ToroCraft/ToroTraits.git
cd ToroTraits
gradle setupDecompWorkspace
```

To setup an Intellij environment:
```
gradle idea
```

To setup an Eclipse environment:
```
gradle eclipse
```
